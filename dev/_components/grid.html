--- layout: component title: Responsive Grid group-name: Grid group-id: 02 description: | Here are some options. __None-responsive:__ The grid system is based on fractions. You can specify a column width by adding a class of .column__c-of-20; c is any number between 1 and 20. __Responsive__: There are 5 options for a responsive grid. Use column without any modifier for mobile size up to small. small: this is for small devices. You set using the brakpoint mixin. This has 20 columns. EG: .column__20-of-20--small This will give a 100% element in the small size breakpoint medium: this is for medium devices. You set using the brakpoint mixin. This has 20 columns. large: this is for large devices (Laptops etc.). You set using the brakpoint mixin. This has 20 columns. extra-large: this is for extra-large devices (27" imac for example). You set using the brakpoint mixin. This has 20 columns. ---
